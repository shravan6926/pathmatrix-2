# 🚀 PathMatrix

PathMatrix is a web-based intelligent route planning application developed to solve two optimization problems:

- 🛣️ Sightseeing Route Optimization using **Beam Search + Simulated Annealing**
- 🚖 Dynamic Ride Sharing Route Planning

The project provides an interactive interface for entering locations, distance matrices, travel constraints, and visualizing optimized routes.

---

## ✨ Features

- 📍 Route Planning
- 🚖 Dynamic Ride Sharing
- 🌍 Interactive Route Visualization
- 📊 Distance Matrix Support
- 📌 Euclidean Coordinate Support
- ⚡ Beam Search Algorithm
- 🔥 Simulated Annealing Optimization
- 🎯 Category Constraints
- 📈 Score Maximization
- 💻 Responsive Modern UI

## 🚀 Installation

Clone the repository

```bash
git clone https://github.com/shravan6926/pathmatrix.git
```

Go to the project folder

```bash
cd pathmatrix
```

Install dependencies

```bash
npm install
```

Run the development server

```bash
npm run dev
```

---

## 🏗️ Build

```bash
npm run build
```

Preview production build

```bash
npm run preview
```

## 🎯 Algorithms Used

### Part A

Hybrid Beam Search + Simulated Annealing

Objective:
- Maximize sightseeing score
- Stay within distance budget
- Satisfy category constraints

### Part B

Dynamic Ride Sharing

Objective:
- Assign passengers efficiently
- Optimize vehicle routes
- Reduce travel cost
